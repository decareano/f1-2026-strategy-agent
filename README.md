# 🏎️ F1 2026 Strategy Agent

An AI-powered strategist designed for the 2026 Formula 1 regulations. This project combines a high-fidelity physics engine with an LLM agent to simulate and optimize race strategy.

## 🚀 Features
* **Physics Engine:** Real-time calculation of fuel flow (MJ/h), ERS deployment, and tyre degradation.
* **Agentic Logic:** A reasoning-driven agent that manages pit stop windows and energy deployment.
* **2026 Spec:** Built specifically around the upcoming technical regulations
